# 📁 Profile Images

Drop Rishabh's real portrait photo here.

## Expected file:

| Filename                  | Used In                          | Size Recommendation |
|---------------------------|----------------------------------|---------------------|
| `rishabh-portrait.jpg`    | Home Hero + About page portrait  | 800×1000px minimum  |

## How to activate:

1. Add your photo file here as `rishabh-portrait.jpg`
2. Open `src/lib/imageConfig.ts`
3. Change the `portrait` value to `/images/profile/rishabh-portrait.jpg`

That's it — both the Home and About pages will instantly use the real photo.
