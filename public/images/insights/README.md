# 📁 Insights / Blog Images

Drop blog post and article images here.

## Expected files:

| Filename                  | Used In                              | Recommended Size     |
|---------------------------|--------------------------------------|----------------------|
| `featured-article.jpg`    | Insights page — featured article     | 1200×675px (16:9)    |

## How to activate:

1. Add the image file here with the exact filename above
2. Open `src/lib/imageConfig.ts`
3. Update `insights.featured` to `/images/insights/featured-article.jpg`
